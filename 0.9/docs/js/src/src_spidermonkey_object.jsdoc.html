  8 /**
  9  * Define a property on an object with flags. This allows you to create
 10  * read-only or non-enumerable properties on your javascript objects.
 11  * @name defineProperty
 12  * @methodOf Object
 13  *
 14  * @param obj Object on which to define the property
 15  * @param name property name
 16  * @param description Property descriptor.
 17  *
 18  * A property descriptor is an object with some of the following keys
 19  *
 20  * * enumerable - is the property visible in `for in` loops
 21  * * configurable - can further changes be made to this property
 22  * * writable - read only property
 23  * * value - initial property value
 24  * * getter - getter function
 25  * * setter - setter function
 26  *
 27  * All values default to false or undefined. If either writable or value exist
 28  * then neither getter nor setter are allowed. 
 29  *
 30  * A getter function is one which is called and should return the current value
 31  * of the property. A setter is called with the argument to set the property
 32  * too, but you can set it do a derivative of that value, or ignore it
 33  * completly. Both getter and setter are called with "this" set to the object
 34  * in question.
 35  *
 36  * {{{
 37  * function MyObj() {
 38  *   var prop = 0;
 39  *   Object.defineProperty(
 40  *     myobj, 
 41  *     "prop",
 42  *     { getter: function() { return prop },
 43  *       setter: function(x) { prop = x * 2; return prop; },
 44  *       enumerable: false,
 45  *       configurable: true
 46  *     }
 47  *   );
 48  * }
 49  * }}}
 50  * 
 51  * If configurable is false, then any further attempts to call
 52  * [[#defineProperty]] on that property will result in an exception.  
 53  */
 54
