    class cURL.Easy

    Description

    Provides a synchronous and quick interface.

    See cURL’s easy Interface if you are curious about the underlying C API.

    Instance properties

    • options #

      cURL.Easy#options -> EasyOpt

      The options property is used to set connection options (see curl_easy_setopt).

      Example

        c.options.url = 'http://www.google.com'

      Options

      The following options are supported. See the libcurl documentation links. Flag parameters are class constants of cURL without the leading CURL(_). (e.g. CURLPROTO_HTTP -> cURL.PROTO_HTTP)

      • integer: boolean or number
      • string: string
      • list: array of strings
      BEHAVIOR OPTIONS

      • verbose, integer (CURLOPT_VERBOSE)

      • header, integer (CURLOPT_HEADER)

      • noprogress, integer (CURLOPT_NOPROGRESS)

      • nosignal, integer (CURLOPT_NOSIGNAL)

        CALLBACK OPTIONS

      • writefunction, function(buffer,size). CURLOPT_WRITEFUNCTION Callback should except buffer and size as parameter. Buffer is a binary.ByteArray and size the size of each object. Should return the length of the buffer if everything is okay.

      • readfunction, function(buffer,size,nmemb). CURLOPT_READFUNCTION

      • seekfunction, function(offset,origin). CURLOPT_SEEKFUNCTION

      • progressfunction, function(dltotal,dlnow,ultotal,ulnow). CURLOPT_SEEKFUNCTION

      • headerfunction, function(data,size). CURLOPT_HEADERFUNCTION

      • debugfunction, function(infotype,buffer). CURLOPT_DEBUGFUNCTION

      ERROR OPTIONS

    Instance methods

    • cleanup #

      cURL.Easy#cleanup() ⇒ undefined

      Destroy cURL object and free resources. cURL objects are cleaned up when garbage collected. You do not need to call this manually.

      See curl_easy_cleanup.

    • escape #

      cURL.Easy#escape(String) ⇒ String

      URL encodes the given string.

      See curl_easy_escape and cURL.Easy#unescape.

    • perform #

      cURL.Easy#perform() ⇒ undefined

      Perform the file transfer. Throws an exception on error.

      See curl_easy_perform.

    • reset #

      cURL.Easy#reset() ⇒ undefined

      Reset options to initial defaults.

      See curl_easy_reset.

    • unescape #

      cURL.Easy#unescape(String) ⇒ String

      URL decodes the given string.

      See curl_easy_unescape and cURL.Easy#escape.

    • valid #

      cURL.Easy#valid() ⇒ Boolean

      Returns true if the object is valid.

      See cURL.Easy#cleanup.
