    class xml.HTMLParser

    Description

    Parse HTML files.

    Do our best to parse tag-soup like html files. Parsed documents are output as XHTML.

    Methods

    • parse
    • parseString

    Class methods

    • parse #

      xml.HTMLParser.parse(source) ⇒ xml.Document
      • source (String | io.Stream) – document to parse

      Parse source, which can either be a filename, a URL or an open file stream; as HTML, returning a valid XHTML document.

    • parseString #

      xml.HTMLParser.parseString(literal) ⇒ xml.Document
      • literal (String) – text to parse

      Parse literal as HTML returning a valid XHTML document.
